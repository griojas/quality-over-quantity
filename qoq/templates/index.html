The service provides an easy way to link and overview Github projects with Circle CI builds. With such stats we review the quality state over time per branch.

Further data agreggation between Github and CircleCi could show estimates of time and effort spent across projects/repositories within an organisation.

Make sure to have ready your Github personal access token. If you dont have one get more information in Github Help

Similarly, make sure you have your Circle CI token handy. Otherwise get more information in Circle CI Help


To see the API docs go to API Documentation

The documentation site allows to test the endpoints, to see the operations available drill down in the default namespace
